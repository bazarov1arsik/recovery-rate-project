## Краткое описание

Программный конвейер на Python для вычисления **recovery rate** — доли callable-области генома,
покрытой хотя бы одним неандертальским сегментом в популяции. Включает пилотный анализ
22-й хромосомы и полногеномное сравнение популяций GBR (британцы) и TSI (тосканцы)
по данным проекта 1000 Genomes и предсказаниям метода DAIseg.

## Основные результаты

- **Пилот (chr22, CEU и IBS):** recovery rate 0.20% (46666 п.н.)
- **Полный геном (91 GBR, 107 TSI):**
  - GBR: recovery rate = 32.86%, средняя индивидуальная примесь = 3.63%
  - TSI: recovery rate = 35.93%, средняя индивидуальная примесь = 3.56%
- **Выровненные выборки (91 GBR, 91 TSI):**
  - GBR: recovery rate = 32.86%
  - TSI (balanced): recovery rate = 34.83%
  - Разница сохраняется (ΔR = 1.96 п.п.), то есть не объясняется размером выборки
- **Бутстрап (B=1000):** 95% доверительный интервал для разности [0.40%, 2.42%]
- **Перестановочный тест (B=10000):** p-value = 0.0001
- **Кривые насыщения:** n95 = 31 (GBR), n95 = 34 (TSI)
- **Энтропийный индекс и геномный профиль:** более равномерное распределение фрагментов у TSI

Все случайные процедуры фиксированы (seed = 42), результаты воспроизводимы.

## Структура репозитория

- `results/` – графики (кривые насыщения, геномный профиль)
